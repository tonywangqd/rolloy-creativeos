## Responsive Behavior

### Desktop (lg: 1024px+)
- Session list in left sidebar (fixed width)
- ABCD selector below sessions
- Main content area 2/3 width

### Tablet (md: 768px - 1023px)
- Session list collapsible
- Stack vertically when expanded
- Main content full width when list collapsed

### Mobile (sm: < 768px)
- Session list as drawer (slide from left)
- Full-width layout
- Touch-optimized session items

## Accessibility Features

### Keyboard Navigation
- `Tab`: Navigate between sessions
- `Enter`: Select session
- `Escape`: Collapse session list
- `ArrowUp/Down`: Move through list

### Screen Reader Labels
```html
<button aria-label="Load session: Beach Sunrise Campaign">
  ...
</button>

<div role="progressbar" aria-valuenow="60" aria-valuemin="0" aria-valuemax="100">
  60%
</div>
```

### Color Contrast
All text meets WCAG AA standards:
- Primary text: 16:1 ratio
- Secondary text: 7:1 ratio
- Muted text: 4.5:1 ratio
